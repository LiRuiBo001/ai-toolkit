## 🚀 快速开始

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/LiRuiBo001/ai-toolkit.git
cd ai-toolkit

# 直接用浏览器打开
open public/index.html          # macOS
start public/index.html         # Windows
```

或者用任何 HTTP 服务器：

```bash
# Python
python -m http.server 8080 -d public

# Node.js
npx serve public

# PHP
php -S 0.0.0.0:8080 -t public
```

然后访问 `http://localhost:8080` 即可。

## 🌍 国际化

站内完整支持中英文切换，切换方式：

- **页面右下角语言切换按钮** — 一键切换
- **URL 参数** — 添加 `?lang=en` 直接加载英文版
- **自动检测** — 读取浏览器语言偏好

所有翻译键集中在 `public/i18n.js`，新增工具只需添加对应的 `zh` / `en` 条目。
